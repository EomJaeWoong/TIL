# 마크다운

1. 마크다운이란?

   - 개요
     - 2004년 존그루버에 의해 만들어 짐
     - md 확장자를 가진 파일
     - 텍스트 기반의 마크업 언어
   - 특징
     - 쉽게 쓰고 읽을 수 있다
     - 별도의 도구 없이 작성이 가능하다
     - 빠르게 컨텐츠 제작이 가능하다
     - 텍스트로 저장되여 용량이 적다
     - 텍스트라 프로그램을 이용해 버전 관리가 용이하다
     - 지원되는 프로그램, 플랫폼이 다양하다

2. 마크업과의 차이

   - Mark up

     - 마크(태그 등)으로 감싼다

       - ex )

         # Hello!

     - 다른 기종과의 데이터를 수월하게 교환하기 위해 사용(HTML)

   - Mark down

     - 표준이 없다
     - 모든 마크업 기능을 대신하지 못한다


     **두껍게**, **두껍게**

   - 목록

     ```
     1. 순서가 필요한 목록
     1. 순서가 필요한 목록
        * 순서가 필요하지 않은 목록(서브)
        * 순서가 필요하지 않은 목록(서브)
     1. 순서가 필요한 목록
        1. 순서가 필요한 목록(서브)
        1. 순서가 필요한 목록(서브) 
     1. 순서가 필요한 목록
     
     - 순서가 필요하지 않은 목록에 사용 가능한 기호
       - - 대쉬
       - * 별표
       - + 더하기
     ```

     1. 순서가 필요한 목록
     2. 순서가 필요한 목록
        - 순서가 필요하지 않은 목록(서브)

     3. 순서가 필요한 목록
        1. 순서가 필요한 목록(서브)
        2. 순서가 필요한 목록(서브)
     4. 순서가 필요한 목록

     - 순서가 필요하지 않은 목록에 사용 가능한 기호
       - **-** 대쉬
       - ***** 별표
       - **+** 더하기
