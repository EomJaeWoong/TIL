# Algorithm

### 하샤드 수

> 문제 링크 : https://programmers.co.kr/learn/courses/30/lessons/12947



###### 풀이

* 각 자리수를 더하여 나누어 떨어지는지 보면 된다

  ```c++
  #include <string>
  #include <vector>
  
  using namespace std;
  
  bool solution(int x) {
      int hashad = 0;
      int num = x;
      
      while(num > 0){
          hashad += num % 10;
          num = num / 10;
      }
      
      return x % hashad == 0? true : false;
  }
  ```
