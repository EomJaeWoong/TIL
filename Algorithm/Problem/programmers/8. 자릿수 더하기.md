# Algorithm

### 자릿수 더하기

> 문제 링크 : https://programmers.co.kr/learn/courses/30/lessons/12931



###### 풀이

* 10으로 나눈 나머지를 더하면 됨

  ```c++
  #include <iostream>
  
  using namespace std;
  int solution(int n)
  {
      int answer = 0;
  
      while(n > 0){
          answer += n % 10;
          n = n / 10;
      }
  
      return answer;
  }
  ```
