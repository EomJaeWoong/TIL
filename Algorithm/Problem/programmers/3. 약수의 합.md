# Algorithm

### 약수의 합



> 문제 링크 : https://programmers.co.kr/learn/courses/30/lessons/12928?language=cpp





###### 풀이

```c++
#include <string>
#include <vector>

using namespace std;

int solution(int n) {
    int answer = 0;
    
    for(int i = 1; i <= n; i++){
        if(n % i == 0)
            answer += i;
    }
    
    return answer;
}
```



```python
def solution(n):
    answer = 0
    
    for e in range(1, n+1) :
        if not n % e :
            answer += e;
    
    return answer
```
