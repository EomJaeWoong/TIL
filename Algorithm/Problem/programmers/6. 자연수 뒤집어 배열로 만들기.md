# Algorithm

### 자연수 뒤집어 배열로 만들기

> 문제 링크 : https://programmers.co.kr/learn/courses/30/lessons/12932



###### 풀이

* 나머지를 배열에 넣고 몫을 대상으로 하여 반복하면 됨

  ```c++
  #include <string>
  #include <vector>
  
  using namespace std;
  
  vector<int> solution(long long n) {
      vector<int> answer;
      
      while(n > 0){
          answer.push_back((int)(n % 10));
          n = n / 10;
      }
      
      return answer;
  }
  ```
